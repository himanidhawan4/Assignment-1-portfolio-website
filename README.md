# 👩‍💻 My First Portfolio Website

Welcome to my first web development project! This is a personal portfolio website designed to showcase my journey as a **B.Tech Computer Science student**. It is a fully responsive, multi-section site built using fundamental front-end technologies.

 
## ✨ Features
* **Responsive Design:** Implemented CSS Media Queries to ensure the layout adapts to mobile, tablet, and desktop screens.
* **Smooth Navigation:** Includes a sticky-style navbar with smooth-scroll linking to About, Projects, and Contact sections.
* **Project Showcase:** A dedicated grid layout for recent work, including details on my first portfolio build.
* **Registration Form:** A functional UI for a contact/registration form with custom-styled inputs.
* **Modern Typography:** Integrated Google Fonts including *Playfair Display*, *Henny Penny*, and *Dancing Script* for a personalized look.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for accessibility and SEO.
* **CSS3:** Custom layouts using **Flexbox**, hover animations, and responsive breakpoints.
* **FontAwesome:** Integrated icons for professional social media linking.
